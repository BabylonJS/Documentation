## Creating Sprites The Easy Way

Babylon.js has a sprite system with many features and components. Babylon.js 4.2 introduced a new tool to make it simple and easy to leverage this powerful system. The Babylon.js Sprite Editor. Built right into [The Inspector](/features/toolsAndResources/tools/inspector), creating sprite managers and controlling sprites is now as easy as right-clicking!
