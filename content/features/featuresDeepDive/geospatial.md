## Geospatial

Babylon.js provides a set of features for building geospatial and large-world 3D applications.

- **[Geospatial Camera](/features/featuresDeepDive/geospatial/geospatialCamera)** - A camera designed for map-like navigation around a spherical planet.
- **[Large World Rendering](/features/featuresDeepDive/scene/large_world)** - Techniques for rendering scenes with very large world coordinates while avoiding floating-point precision issues.
- **[Loading 3D Tiles](/features/featuresDeepDive/geospatial/loading3dTiles)** - An overview of the 3D Tiles format and how to load 3D Tiles datasets in Babylon.js.
